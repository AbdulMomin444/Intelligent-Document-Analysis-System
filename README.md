# Intelligent Document Analysis System

## 📌 Project Description
This project is an AI-based system that analyzes scanned documents using Computer Vision and OCR techniques. It extracts text, detects document structure, and converts it into structured formats.

## 🚀 Features
- Image preprocessing using OpenCV
- Text extraction using OCR (Tesseract)
- Document layout detection
- Structured data output

## 🛠️ Technologies
- Python
- OpenCV
- Tesseract OCR
- Scikit-learn

## 📂 Project Structure
- code/ → main implementation
- dataset/ → input images
- outputs/ → results
